# Product Roadmap

## Purpose

This roadmap breaks the MVP into milestones and sprints. Complete each
sprint before starting the next.

-   Configure flavors (optional)
-   Setup Git repository
-   Connect Supabase
-   Connect Firebase

-   Setup environments (.env)
